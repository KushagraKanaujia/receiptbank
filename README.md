### Prerequisites

### Installation

\`\`\`bash
# Clone the repository
git clone https://github.com/yourusername/datahub.git
cd datahub

# Backend setup
cd backend
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev

# Frontend setup (new terminal)
cd frontend
npm install
npm run dev
